# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-04

### Added
- Logo assets (jks-logo-w-type.svg and jks-logo-no-type.svg) for branding
- Unique flowing abstract background pattern across all hero sections
- About and Contact pages with consistent design language
- Social media icons in portfolio cards (Languages 4 and VISUALIZE SPORTS)
- Envelope icon for Contact page visual interest
- Orange (#e07b5a) accent color for CTA sections and page headings

**SEO Enhancements:**
- Open Graph and Twitter Card meta tags for social media sharing
- JSON-LD schema.org structured data for organization information
- robots.txt file for search engine crawling guidelines
- sitemap.xml for all website pages
- Enhanced page titles with keyword optimization
- Theme color meta tag for browser UI consistency
- Author and language meta tags
- Apple touch icon support
- Comprehensive SEO documentation (SEO.md)

### Changed
- Updated all icon and link colors from blue (#0099ff) to brand teal (#183e4b)
- Improved hero section design with custom SVG wave patterns
- Enhanced page hierarchy with orange H1 headings on About and Contact pages
- Refined spacing and layout across all pages
- Improved text capitalization and consistency (e.g., "Get in Touch")
- Page titles now include relevant keywords for better search visibility

### Fixed
- Social media icon colors now consistent across all pages
- Text and icon styling aligned with brand color palette
- Hero sections now include unique flowing abstract backgrounds
- Canonical URLs added to prevent duplicate content issues
